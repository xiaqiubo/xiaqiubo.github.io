windows下配置composer

发布 : 2016-11-12 分类 : 笔记

1. 配置php环境

我使用的是phpstudy其他环境除路径外，其他类似

  1. 我的电脑 –> 属性 –> 高级系统设置 –> 高级 –> 环境变量

  2. 在系统变量中找到path变量 –> 编辑其值 –> 添加路径 c:\\phpstudy\\php55n;

    注：此步骤注意最后的分号（英文）不能少

  3. 在系统变量中新建

    变量名：`PHP_HOME`
值：`c:\phpstudy\php55n`
    此步骤不需要分号

  4. 配置检测

    1. 打开cmd dos命令行

      win + R 打开运行  -->  cmd

    2. 输入 php -v

      显示 php 5.5.30 ·····

    3. php环境变量配置成功

2. 配置composer
