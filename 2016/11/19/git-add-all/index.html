【小码农博客】git添加文件夹下所有文件

发布 : 2016-11-19 分类 : 笔记

最近，突然想将之前做的部分项目上传至 github 上，却发现网上对于上传整个文件夹的讲解多种多样，转念一想，其实和添加文件是一样的，首先打开git-bash命令行

git status

发现有一个新的文件夹 添加文件是

git add (files name)

或者添加文件夹下所有的文件所

git add dirname/*

这样一下所有的文件都被添加到 git 暂存区了， 剩下的就是 git commit 和 git push了。
